## 运行方式

修改 `model` 和 `file`，选择使用的模型和数据集。

在 Chatbot 类的 complete 函数中修改不同模型获取回复的方式。

```python
python main.py
```

## 意外中断处理

程序退出时会保存已生成的数据，运行时会读取 `save_path` 并加载已生成的数据，从未生成的数据处继续。

## 数据集固定下更换模型时的处理

数据集固定时，在使用一个模型生成完数据后，若更换模型，需要原数据替换为 new 文件夹中的数据，new 文件夹数据需清空。

建议的流程是：使用单个模型运行完所有数据集；原数据替换为 new 文件夹中的数据，清空 new 文件夹数据；更换模型继续运行。
